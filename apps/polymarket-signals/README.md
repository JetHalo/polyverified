# Poly Verified

Polymarket signals UI built with Vite, React, TypeScript, shadcn-ui, and Tailwind CSS.

## Development

```sh
cd /Users/jethalo/projects/zkpoly/polymarket-signals
npm install
npm run dev
```

The local dev server runs on `http://localhost:8080`.

## Build

```sh
cd /Users/jethalo/projects/zkpoly/polymarket-signals
npm run build
```

## Test

```sh
cd /Users/jethalo/projects/zkpoly/polymarket-signals
npm test
```
